
# Coding Challenge

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Tech Stack
- **Frontend:** Next.js (React), TypeScript, CSS (TailwindCSS)

## Features
- Task List for several Categories
- Editable Categories and Task Descriptions
- Timer for urgency of tasks
- local storage persistence

## Getting Started

1.) Clone Repository:
```bash
git clone https://github.com/KatFoxx/CodingChallenge.git
cd CodingChallenge
```
2.) Install dependencies:
```bash
npm install
```
3.) Run development server:
```bash
npm run dev
```
4.) Open in browser:
http://localhost:3000
